Features

Fully Customisable
Customise almost any aspect of the cropper component, or use it right out of the box.
Mobile Support
Vue Advanced Cropper supports desktop and mobile devices alike. Built in support for touch events, cropper resizing, etc.
Canvas / Coordinates
This Cropper can be used to create a canvas with a cropped area or just coordinates relative to the original image to crop it server-side in the future.
Advanced Features
Set minimum and maximum aspect ratios, customizie minimum and maximum height and width, etc.

How to start?

This library gives you numerous possibilities to customize your cropper. It's almost the framework for creating of the croppers. But also it's very easy to use. Install the library and use the fragments of code below. That's all that you need to start.
/
$yarn add react-advanced-cropper

import React, { useState } from 'react';import { CropperRef, Cropper } from 'react-advanced-cropper';import 'react-advanced-cropper/dist/style.css'
